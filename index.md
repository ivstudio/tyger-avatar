# tyger-avatars

> An Illustrated avatar react library

## Installation

```
npm i tyger-avatars
```

## Usage

```js
import { TrChelsea } from 'tyger-avatars';

function App() {
	return (
		<>
			<TrChelsea style={{ fontSize: '40px' }} />
		</>
	);
}
```

## options

```js
<TrSamantha
	style={{ fontSize: '80px' }}
	title="my title"
	titleId="my-title-id"
/>
```
