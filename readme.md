> An illustrated avatar react library

[View all avatars](https://ivstudio.github.io/tyger-avatar-wiki/)

---

## Installation

```bash
npm i tyger-avatar
```

## Usage

```jsx
import TygerAvatar from 'tyger-avatar';

<TygerAvatar name="TrIggy" />;
```
