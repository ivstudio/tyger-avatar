## Installation

```bash
npm i tyger-avatar
```

## Usage

```jsx
import { TrChelsea } from 'tyger-avatar';

<TrTorsten width="120" />;
```

## options

```js
import { TrSamantha } from 'tyger-avatar';

<TrSamantha width="40" title="my title" titleId="my-title-id" />;
```
