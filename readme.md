# tyger-avatar

> An illustrated avatar react library

## Installation

```bash
npm i tyger-avatar
```

## Usage

```jsx
import { TrChelsea } from 'tyger-avatar';

const App = () => <TrChelsea style={{ fontSize: '40px' }} />;
```

## options

```js
<TrSamantha
	style={{ fontSize: '80px' }}
	title="my title"
	titleId="my-title-id"
/>
```
